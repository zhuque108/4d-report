## 1. 产品概述

### 1.1 产品背景

Y-CAS（Yikangshun Clinical Assessment System）是基于《儿科学》及《中华儿科杂志》临床共识构建的儿童身高管理评估系统。系统采用4D-Growth生长优化模型，从深度睡眠（D1）、精准营养（D2）、纵向运动（D3）、情绪与习惯（D4）四个维度进行系统评估。

### 1.2 产品定位

- **目标市场**：0-18岁儿童身高管理领域
- **核心价值**：科学评估、个性化干预、持续跟踪
- **差异化优势**：医育结合、4D科学模型、可视化报告

### 1.3 产品目标

1. 建立标准化的儿童身高评估流程
2. 生成专业、美观的个性化评估报告
3. 提供科学的干预建议和跟踪方案
4. 支持机构端和家长端双端使用

#### 3.2.2 4D维度评估录入

**功能描述**：
- 通过问卷形式收集4D维度数据
- 支持逐项评分和自动汇总
- 提供评估指引和说明

**D1 - 深度睡眠评估**：

| 评估项 | 选项 | 分值 | 权重 |
|--------|------|------|------|
| 入睡困难程度 | 无/偶尔/经常/总是 | 25/20/10/0 | 25% |
| 夜间觉醒次数 | 0次/1-2次/3次以上 | 25/15/5 | 25% |
| 早醒情况 | 无/偶尔/经常 | 25/15/5 | 25% |
| 日间精神状态 | 充沛/一般/疲倦 | 25/15/5 | 25% |
| 实际入睡时间 | 21:30前/22:00前/22:00后 | +5/+0/-5 | 附加 |
| 睡眠时长 | 达标/基本达标/不足 | +5/+0/-5 | 附加 |

**D2 - 精准营养评估**：

| 评估项 | 选项 | 分值 | 权重 |
|--------|------|------|------|
| 挑食偏食程度 | 无/轻度/中度/重度 | 25/20/10/0 | 25% |
| 进餐规律性 | 规律/较规律/不规律 | 25/15/5 | 25% |
| 零食饮料摄入 | 很少/偶尔/经常/频繁 | 25/20/10/0 | 25% |
| 早餐习惯 | 每天吃/偶尔不吃/经常不吃 | 25/15/5 | 25% |
| 蛋白质摄入 | 达标/不足 | +5/-5 | 附加 |
| 钙质摄入 | 达标/不足 | +5/-5 | 附加 |

**D3 - 纵向运动评估**：

| 评估项 | 选项 | 分值 | 权重 |
|--------|------|------|------|
| 运动频率 | ≥5次/3-4次/≤2次/周 | 25/20/10 | 25% |
| 纵向运动占比 | ≥60%/40-60%/<40% | 25/15/5 | 25% |
| 单次运动时长 | ≥45min/30-45min/<30min | 25/15/5 | 25% |
| 运动强度 | 中等/轻度/剧烈 | 25/15/10 | 25% |
| 运动时段 | 最佳时段/一般时段 | +5/+0 | 附加 |

**D4 - 情绪与习惯评估**：

| 评估项 | 选项 | 分值 | 权重 |
|--------|------|------|------|
| 焦虑情绪 | 无/偶尔/经常/总是 | 25/20/10/0 | 25% |
| 睡眠质量（情绪相关） | 好/一般/差 | 25/15/5 | 25% |
| 食欲变化 | 无变化/轻微/明显 | 25/20/10 | 25% |
| 社交意愿 | 高/中/低 | 25/15/5 | 25% |
| 家庭氛围 | 和谐/一般/冲突 | +5/+0/-5 | 附加 |
| 父母期望 | 合理/较高/过高 | +5/+0/-5 | 附加 |

#### 3.2.3 数据验证与质量控制

**验证规则**：
1. **逻辑校验**：身高体重是否在合理范围
2. **趋势校验**：与历史数据对比，异常波动提示
3. **完整性校验**：必填项完整性检查
4. **重复性校验**：同一日期重复录入提示

**异常提示**：
- 身高增长异常（月增长<0.3cm或>2cm）
- BMI异常（偏瘦或超重）
- 骨龄与实际年龄差异过大（>2年）

#### 3.3.3 报告数据计算逻辑

**评分计算**：
```
维度得分 = Σ(评估项得分 × 权重) + 附加分
综合得分 = Σ(维度得分 × 25%)
```

**评级标准**：
| 得分范围 | 评级 | 颜色标识 |
|----------|------|----------|
| 90-100 | 优秀 | 绿色 #10b981 |
| 80-89 | 良好 | 蓝色 #0c9af0 |
| 60-79 | 一般 | 橙色 #f59e0b |
| <60 | 较差 | 红色 #ef4444 |

**风险等级判定**：
| 风险类型 | 低风险 | 中风险 | 高风险 |
|----------|--------|--------|--------|
| 生长迟缓 | 年增长≥5cm | 3-5cm | <3cm |
| 早熟风险 | 骨龄±1岁 | ±1-2岁 | >±2岁 |
| 营养不良 | BMI正常 | 偏瘦/超重 | 严重偏离 |
| 心理问题 | 无异常 | 偶发 | 持续存在 |

**身高预测模型**：
1. **遗传靶身高**：
   - 男孩 = (父身高 + 母身高 + 13) / 2 ± 5cm
   - 女孩 = (父身高 + 母身高 - 13) / 2 ± 5cm

2. **当前轨迹预测**：基于当前生长曲线外推

3. **4D优化预测**：
   ```
   优化后身高 = 当前轨迹预测 + 优化潜力
   优化潜力 = Σ(维度提升空间 × 影响系数)
   ```

### 6.3 API接口设计

```yaml
# 创建评估
POST /api/v1/assessments
Request:
  child_id: string
  assessment_date: date
  height: number
  weight: number
  d1_data: object
  d2_data: object
  d3_data: object
  d4_data: object
Response:
  assessment_id: string
  assessment_no: string

# 获取评估详情
GET /api/v1/assessments/{id}
Response:
  id: string
  assessment_no: string
  child_info: object
  measurements: object
  scores: object
  risks: object
  predictions: object

# 生成报告
POST /api/v1/assessments/{id}/generate-report
Request:
  template_type: string
Response:
  report_id: string
  pdf_url: string

# 导出PDF
GET /api/v1/reports/{id}/export
Query:
  format: pdf
Response:
  file: binary
```
