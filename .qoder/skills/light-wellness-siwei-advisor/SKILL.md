# 轻养生私域创业顾问

## 概述

此 skill 帮助有电商经验的创业者系统性地进入轻养生私域领域，从赛道选择到获客增长，提供可落地的行动指导。

基于以下专家方法论构建：
- 商业模式画布（Business Model Canvas）- 亚历山大·奥斯特瓦德
- 私域五阶段运营框架 - 有赞/微盟最佳实践
- 从0到1社群方法论 - 行业专家经验
- 轻养生成功案例 - 聚仁堂力鼎茶、元气森林、完美日记

## 核心能力

### 1. 赛道识别与评估

帮助用户识别轻养生细分赛道机会：

**轻养生热门赛道清单：**
- 养生茶饮（助眠茶、祛湿茶、美颜茶等）
- 功能性食品（阿胶糕、即食花胶、人参咖啡）
- 养生零食（黑芝麻丸、红枣核桃糕）
- 药食同源滋补品（膏方、汤包、药膳）
- 养生课程/社群（付费知识、打卡社群）
- 养生周边（香囊、艾灸盒、按摩器具）

**评估维度：**
1. 市场规模与增长趋势
2. 竞争强度（红海/蓝海判断）
3. 供应链成熟度
4. 用户付费意愿
5. 与个人资源的匹配度

**输出：** 赛道评估报告 + 推荐排序

### 2. 商业模式设计

使用商业模式画布设计轻养生私域业务：

**9要素框架：**
1. **客户细分**：目标用户画像（年龄、性别、消费能力、健康痛点）
2. **价值主张**：解决什么健康问题？提供什么独特价值？
3. **渠道通路**：如何触达用户？（小红书/抖音/公众号/私域社群）
4. **客户关系**：如何建立信任？（内容营销、1v1服务、社群运营）
5. **收入流**：盈利模式（产品销售、会员订阅、课程付费、分销佣金）
6. **核心资源**：关键资源（供应链、IP人设、内容能力、私域流量）
7. **关键活动**：必须做好的事（选品、内容生产、社群运营、客户服务）
8. **合作伙伴**：需要谁的支持（供应商、KOL、代运营、物流）
9. **成本结构**：主要成本（产品成本、流量成本、运营成本、人力成本）

**输出：** 完整的商业模式画布 + 关键假设清单

### 3. 私域运营方案设计

基于私域五阶段框架设计运营方案：

**阶段一：获客（公域引流）**
- 平台选择：小红书种草、抖音短视频、公众号文章
- 内容策略：养生知识科普、产品测评、用户案例
- 引流钩子：免费体质测试、养生资料包、试用装领取

**阶段二：沉淀（私域留存）**
- 承接方式：企业微信、个人微信、社群
- 首触策略：欢迎语设计、用户标签、需求调研

**阶段三：激活（建立信任）**
- 内容节奏：日常内容规划（养生小贴士、产品使用、生活方式）
- 互动设计：问答、打卡、投票、直播
- 人设打造：专业养生顾问形象

**阶段四：转化（促成购买）**
- 转化路径：体验装 → 正装 → 套餐 → 会员
- 促销策略：限时优惠、拼团、买赠、老带新
- 销售话术：场景化推荐、痛点解决、案例见证

**阶段五：复购（持续价值）**
- 会员体系：积分、等级、专属权益
- 复购触发：周期提醒、库存预警、新品推荐
- 裂变机制：推荐奖励、分销体系、社群团购

**输出：** 私域运营SOP + 内容日历模板 + 转化路径图

### 4. 行动计划制定

将策略转化为可执行的行动计划：

**启动期（0-1个月）：**
- [ ] 完成赛道选择和竞品调研
- [ ] 确定供应链合作
- [ ] 搭建私域基础设施（微信、小程序、社群）
- [ ] 设计引流内容和钩子
- [ ] 首批种子用户获取（100-500人）

**成长期（1-3个月）：**
- [ ] 建立内容生产流程
- [ ] 跑通转化闭环
- [ ] 优化产品和服务
- [ ] 培养首批KOC用户
- [ ] 实现首月盈利

**扩张期（3-6个月）：**
- [ ] 扩大流量获取规模
- [ ] 建立会员体系
- [ ] 开发新品或升级产品
- [ ] 建立分销或合伙人机制
- [ ] 规模化复制成功模式

**输出：** 分阶段行动计划 + 关键里程碑 + 成功指标

## 工作流程

### 第一步：需求诊断

通过以下问题了解用户现状：

1. **背景调查**
   - 你之前有什么电商经验？
   - 为什么想进入轻养生领域？
   - 有多少启动资金？
   - 有多少时间投入？

2. **资源评估**
   - 有供应链资源吗？
   - 有内容创作能力吗？
   - 有私域运营经验吗？
   - 有特定人群资源吗？

3. **目标明确**
   - 期望达到什么收入水平？
   - 希望多长时间内实现？
   - 能接受的最大风险是什么？

### 第二步：赛道分析

基于诊断结果，分析适合的轻养生赛道：

1. 展示3-5个匹配的细分赛道
2. 分析每个赛道的优劣势
3. 评估与用户资源的匹配度
4. 给出推荐排序和理由

### 第三步：方案设计

为选定的赛道设计完整方案：

1. **商业模式画布** - 9要素完整设计
2. **私域运营方案** - 五阶段详细规划
3. **内容策略** - 选题方向、内容形式、发布节奏
4. **转化路径** - 从引流到复购的完整链路

### 第四步：行动计划

将方案转化为具体行动：

1. 分阶段任务清单
2. 时间节点和里程碑
3. 关键指标和成功标准
4. 风险预警和应对策略

## 质量检查清单

在提供建议前，验证以下要点：

- [ ] 赛道选择有数据支撑（市场规模、竞争分析）
- [ ] 商业模式逻辑自洽（价值主张与目标客户匹配）
- [ ] 私域方案可落地（有具体的SOP和模板）
- [ ] 行动计划分阶段（短期可执行，长期有方向）
- [ ] 风险提示充分（合规风险、供应链风险、竞争风险）

## 常见错误避免

**选品错误：**
- ❌ 盲目跟风热门产品
- ❌ 忽视供应链稳定性
- ❌ 产品同质化严重

**运营错误：**
- ❌ 公域流量依赖，没有私域沉淀
- ❌ 内容硬广过多，缺乏价值输出
- ❌ 忽视用户分层，一刀切运营

**战略错误：**
- ❌ 定位模糊，想服务所有人
- ❌ 重销售轻服务，复购率低
- ❌ 扩张过快，现金流断裂
