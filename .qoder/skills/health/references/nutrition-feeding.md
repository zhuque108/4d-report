# 儿童营养与喂养参考

## 2024 中国婴幼儿膳食指南核心建议

### 0-6个月婴儿
- **喂养原则**：纯母乳喂养，不添加水和其他食物
- **维生素D**：每日补充400国际单位
- **钙补充**：母乳喂养儿无缺钙表现一般不需额外补钙
- **开奶时间**：尽早开奶（出生后1小时内）
- **回应式喂养**：观察饥饿信号（觅食、吸吮手指），不强迫进食

### 6-24个月辅食添加
- **添加时机**：满6个月开始（早产儿按矫正月龄4-5个月）
- **添加原则**：
  - 从单一食物开始，逐样尝试
  - 由稀到稠、由细到粗
  - 每次只添加一种新食物，观察3-5天
  - 优先添加富含铁的食物（强化铁米粉、肉泥）
- **食物禁忌**：1岁内不加盐、糖及调味品
- **奶量保持**：继续母乳/配方奶，辅食是补充而非替代

### 2-3岁幼儿
- **膳食安排**：合理膳食，培养良好饮食行为
- **食物选择**：新鲜、营养丰富食材，少盐少糖
- **进餐习惯**：
  - 进餐时不看电子产品
  - 每次进餐时间控制在30分钟以内
  - 鼓励自主进食

## 婴幼儿膳食宝塔（五层结构）

| 层级 | 食物类别 | 主要营养 |
|-----|---------|---------|
| 塔尖 | 油脂和糖 | 能量，需控制 |
| 第四层 | 肉类、鱼虾、蛋类 | 优质蛋白质、铁、锌 |
| 第三层 | 奶类和大豆类 | 钙、优质蛋白质 |
| 第二层 | 蔬菜和水果 | 维生素、矿物质、膳食纤维 |
| 底层 | 谷物类 | 主要能量来源 |

## 营养评估时间节点

婴幼儿期共进行 **9次营养喂养评估**：
- 月龄：1、3、6、8、12、18、24、30、36个月
- 内容：健康教育、喂养行为评价、营养状况评价、咨询指导

## 常见营养问题识别

### 缺铁性贫血
- 高发年龄：6-24个月
- 预防：及时添加富铁辅食
- 信号：面色苍白、易疲劳、食欲差

### 营养不良分级
| 类型 | 轻度 | 中度 | 重度 |
|-----|-----|-----|-----|
| 低体重 | 均值-2SD至-1SD | 均值-3SD至-2SD | <均值-3SD |
| 生长迟缓 | <均值-2SD | - | - |
| 消瘦 | <均值-2SD | - | - |

### 超重/肥胖评估
- 计算BMI：体重(kg) / 身高(m)²
- 超重：BMI > P85
- 肥胖：BMI > P97

## 喂养困难识别
- 进食时间过长（>40分钟）
- 明显挑食/偏食（食物种类<20种）
- 生长曲线下降
- 进食时频繁哭闹
