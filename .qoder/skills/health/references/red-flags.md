# 儿童健康危险信号（RED FLAGS）

## 发热儿童严重疾病警示分级系统

### 红区（高危）- 立即就医
存在任一症状即属高危：
- [ ] 嗜睡或昏迷
- [ ] 无法安抚的持续性哭闹
- [ ] 呼吸困难或呼吸急促
- [ ] 紫绀（口唇、指甲发紫）
- [ ] 抽搐
- [ ] 颈部僵硬
- [ ] 皮肤瘀点/瘀斑
- [ ] 脱水征象（无泪、尿极少）
- [ ] 无法进食或饮水
- [ ] 体温≥39℃伴精神差

### 黄区（中危）- 尽快就医
存在任一症状需尽快就诊：
- [ ] 发热持续>3天
- [ ] 体温≥39℃但精神尚可
- [ ] 拒食或进食明显减少
- [ ] 活动量明显下降
- [ ] 尿量减少
- [ ] 腹泻伴呕吐
- [ ] 咳嗽伴喘息

### 绿区（低危）- 居家护理
仅有以下表现可居家观察：
- [ ] 体温<39℃
- [ ] 精神状态正常
- [ ] 进食尚可
- [ ] 无呼吸困难
- [ ] 无脱水表现

**绿区需知**：即使低危，如出现症状加重或持续>3天，应转为黄区处理。

## 各年龄段紧急就医指征

### 0-3个月婴儿
以下情况必须立即就医：
- 任何发热（体温≥38℃）
- 拒奶或进食量骤减
- 哭声异常（尖利或微弱）
- 呼吸>60次/分
- 皮肤发灰或发黄加重

### 3-12个月婴儿
- 发热≥39℃持续>24小时
- 精神萎靡或异常烦躁
- 囟门明显膨隆
- 腹泻伴脱水
- 皮疹伴发热

### 1岁以上儿童
- 持续高热>3天
- 剧烈头痛或颈部僵硬
- 呼吸困难
- 腹痛伴呕吐、不排气
- 外伤后意识改变

## 特殊人群预警
以下儿童出现任何不适建议尽早就医：
- 早产儿
- 有基础疾病（哮喘、心脏病等）
- 免疫缺陷
- 营养不良
- 近期手术史

## 家长直觉
如果家长感觉"孩子状态不对"，即使无法准确描述症状，也建议就医评估。家长的直觉往往是重要的预警信号。
