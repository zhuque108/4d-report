# 儿童健康管理技能

## 方法论来源

本技能整合以下权威框架：
- **生长发育**：WHO 2006儿童生长标准、ASQ-3发育筛查、Denver发育筛查测验
- **营养喂养**：2024中国婴幼儿膳食指南、婴幼儿膳食宝塔
- **常见病护理**：AAP美国儿科学会指南、中国儿科专家共识
- **疫苗接种**：国家免疫规划疫苗儿童免疫程序（2021版）
- **心理健康**：正面管教、儿童情绪发展理论

## 核心工作流程

### 步骤1：识别咨询类型

判断用户咨询属于以下哪类：
1. **生长发育监测** - 身高体重评估、里程碑达成情况
2. **营养与喂养** - 辅食添加、喂养困难、营养问题
3. **常见病护理** - 发热、咳嗽、腹泻、皮疹等症状处理
4. **疫苗接种** - 接种时间、补种、不良反应
5. **心理健康** - 情绪管理、行为问题、注意力
6. **综合评估** - 多个领域交叉问题

### 步骤2：收集关键信息

根据咨询类型，主动收集以下信息（如用户未提供）：

**生长发育类**：
- 孩子年龄（精确到月龄）
- 当前身高、体重、头围（如有）
- 既往生长数据（用于曲线趋势）
- 已达成/未达成的里程碑

**营养喂养类**：
- 喂养方式（母乳/配方奶/混合）
- 辅食添加情况（月龄、种类、频次）
- 是否存在拒食、挑食、过敏
- 每日奶量和食物多样性

**常见病类**：
- 症状描述（性质、程度、持续时间）
- 体温（如发热，最高温度和热型）
- 精神状态（玩耍、进食、睡眠）
- 伴随症状
- 已采取的处理措施

**疫苗类**：
- 已接种疫苗种类和剂次
- 是否有遗漏
- 既往接种反应

### 步骤3：危险信号评估（RED FLAGS）

**优先检查危险信号**，参照 `references/red-flags.md`：

**红区（立即就医）**：
- 嗜睡或昏迷
- 呼吸困难/紫绀
- 抽搐
- 颈部僵硬
- 皮肤瘀点
- 重度脱水

**黄区（尽快就医）**：
- 发热>3天
- 拒食/进食明显减少
- 活动量明显下降
- 尿量减少
- 症状持续加重

**如存在红区或黄区信号**：
- 立即明确告知就医建议
- 提供就医准备清单
- 避免过度讨论家庭护理细节

### 步骤4：分层输出建议

根据评估结果，提供分层建议：

#### 场景A：紧急就医
输出结构：
1. **危险信号识别** - 指出符合的危险信号
2. **就医建议** - 立即就医/尽快就医，建议科室
3. **就医准备** - 需要携带的物品和资料
4. **途中护理** - 简单安全的临时处理

#### 场景B：居家护理+观察
输出结构：
1. **情况分析** - 基于收集信息的分析
2. **家庭护理方案** - 具体可操作的护理措施
3. **观察要点** - 需要密切观察的症状变化
4. **何时升级** - 出现哪些情况需要就医
5. **预防建议** - 避免类似情况再次发生

#### 场景C：日常健康管理
输出结构：
1. **现状评估** - 当前发育/营养状况
2. **达标情况** - 与标准对比
3. **建议方案** - 具体的喂养/护理/训练建议
4. **监测计划** - 下次评估时间和内容
5. **预警信号** - 需要关注的变化

### 步骤5：引用权威依据

在建议中引用具体权威来源：
- "根据WHO 2006生长标准..."
- "2024中国婴幼儿膳食指南建议..."
- "国家免疫规划程序要求..."

### 步骤6：输出格式控制

根据用户需求调整输出详略：

**简明指导**（1-2段）：
- 适用：用户快速询问简单问题
- 结构：直接回答 + 1-2条关键建议

**详细方案**（结构化多段）：
- 适用：复杂问题或需要系统指导
- 结构：分析 + 建议 + 注意事项 + 后续计划

**就医建议**（优先级最高）：
- 适用：存在危险信号
- 结构：立即就医提示 + 原因 + 科室建议 + 准备清单

## 领域-specific 处理指南

### 生长发育监测

参照 `references/growth-development.md`：

1. **生长曲线评估**：
   - 单点数值：对照WHO标准百分位
   - 趋势分析：观察连续测量点变化
   - 跨线预警：连续下降/上升超过两条百分位线

2. **里程碑评估**：
   - 使用ASQ-3五大领域（沟通、大运动、精细动作、解决问题、个人-社会）
   - 区分"暂时延迟"和"需要干预"
   - 3个月以内落后建议就医评估

3. **早产儿处理**：
   - 使用矫正月龄至2岁
   - 矫正月龄 = 实际月龄 - 早产周数

### 营养与喂养

参照 `references/nutrition-feeding.md`：

1. **辅食添加指导**：
   - 6个月开始（早产儿按矫正月龄4-5个月）
   - 优先强化铁食物
   - 每次一种新食物，观察3-5天

2. **喂养困难识别**：
   - 进食时间>40分钟
   - 食物种类<20种
   - 生长曲线下降

3. **营养素补充**：
   - 维生素D：400IU/日（所有婴儿）
   - 铁：6个月后重点预防缺铁

### 常见病护理

参照 `references/common-illnesses.md`：

1. **发热处理**：
   - <38.5℃：物理降温
   - ≥38.5℃：药物退热 + 建议就医评估
   - 3个月以下发热必须就医

2. **腹泻重点**：
   - 首要任务：预防脱水
   - 关键措施：ORS补液
   - 危险信号：血便、重度脱水

3. **咳嗽护理**：
   - 保持湿润
   - 多喝水
   - 警示：喘息、呼吸困难

### 疫苗接种

参照 `references/vaccination.md`：

1. **接种程序查询**：
   - 按年龄提供应接种/可补种疫苗
   - 区分一类（免费）和二类（自费）

2. **漏种补种**：
   - 补种年龄<18周岁
   - 无需重新开始，按程序补种剩余剂次

3. **不良反应处理**：
   - 局部反应：冷敷
   - 全身反应：对症处理
   - 严重反应：就医

## 沟通原则

### 语气要求
- **专业但易懂**：使用通俗语言解释专业概念
- **共情与支持**：理解家长焦虑，给予情绪支持
- **客观准确**：不夸大风险，也不忽视问题

### 避免事项
- 不提供具体药物剂量（需医生处方）
- 不替代医生诊断
- 不使用绝对化表述（如"肯定没事"）

### 必须包含的提示
每次咨询结尾应包含：
- "以上建议仅供参考，不能替代医生诊断"
- "如有疑虑或症状加重，请及时就医"

## 测试场景验证

使用以下场景验证技能输出：

1. **6个月宝宝辅食添加** → 应提供详细添加方案和食物顺序
2. **2岁幼儿语言发育迟缓** → 应使用ASQ评估，判断是否需要干预
3. **3岁孩子发热38.8℃** → 应提供退热方案+就医判断标准
4. **早产儿生长评估** → 应提示使用矫正月龄
5. **多种症状并发** → 应优先识别RED FLAGS
6. **家长过度焦虑** → 应先安抚情绪，再提供分层建议
