### 八、高风险预警：这些"产品"需谨慎

| 产品/成分            | 风险描述                                               | 健康师立场                               |
| :------------------- | :----------------------------------------------------- | :--------------------------------------- |
| **"增高药"含性激素** | 某些中药/保健品非法添加睾酮/雌激素，导致骨骺提前闭合   | 严格审查成分，拒绝来源不明产品           |
| **超剂量补钙**       | >2000mg/日钙可能导致肾结石、便秘、铁锌吸收抑制         | 监测血钙、尿钙，提倡食补优先             |
| **超大剂量GABA**     | >500mg/日可能引起嗜睡、注意力涣散、血压波动            | 严守法定上限，睡前服用，观察日间精神状态 |
| **生长激素口服片**   | GH为蛋白质，口服会被消化酶分解，无效且可能存在卫生风险 | 明确告知rhGH必须皮下注射                 |
| **旧标准骨龄片**     | 使用GP图谱法可能误判骨龄，导致错误干预时机             | 坚持使用中华-05或TW3法                   |

**专业提示**：

- **核心逻辑**：身高管理=遗传潜力(70%)×环境因素(30%)，营养补充是环境因素中最易干预的部分，但**不能替代医疗诊断**。
- **监测重于补充**：建立规范的身高体重监测档案（每月固定时间测量）比盲目补充更重要。
- **协同效应**：赖氨酸+GABA+深睡眠+纵向运动产生协同促生长效应，单一成分效果有限。
- **医疗边界**：当年生长速度<4cm、骨龄滞后>2年或超前>1.5年、预测成年身高<-2SD时，必须转介儿科内分泌专科。
